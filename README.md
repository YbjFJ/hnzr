# AI Platform

一个前后端分离的 AI 平台项目。

## 项目结构

```
├── backend/    # 后端 (FastAPI + SQLAlchemy)
├── front/      # 前端 (Vue 3 + Vite + Element Plus)
└── README.md
```

## 技术栈

- **前端**：Vue 3、Vite、Element Plus、Vue Router、Pinia
- **后端**：FastAPI、SQLAlchemy、MySQL

## 快速开始

### 后端

```bash
cd backend
pip install -r requirements.txt
# 配置 .env 后运行
python run.py
```

默认端口：8002

### 前端

```bash
cd front
npm install
npm run dev
```

## 环境要求

- Node.js ^20.19.0 或 >=22.12.0
- Python 3.11+
- MySQL
